〇ゲーム起動方法
1.build and runでexeファイルを生成
2.exeファイルを起動する
3.createAndjoinボタンでルームに入る
4.相手プレイヤーが入ってくるまで待機
※1台のPCで2人分ゲームを立ち上げるにはexeファイルを2つ起動する。

〇ゲームのルール
・ターン制のボードゲーム。相手のゴールにたどりつけば勝ち
・全てのポケモンの初期位置はベンチ。
・ベンチからフィールドに移動するときは必ずエントリーポイントを経由する。
・相手のポケモンが進路を妨害してきたら、バトルを仕掛けてどかせる。
・バトルで気絶したポケモンはP.Cへと送られる。
・バトルはルーレットによって行われる。

〇操作方法
・自分のターンのみ、自分のポケモンの操作が可能
・自分のポケモンをクリックすると移動可能範囲が表示される。
　その状態で範囲内のノードをクリックするとそこに移動する
・相手のポケモンに隣接した状態で相手のポケモンをクリックするとバトルが開始
・移動後に相手のポケモンに隣接していない、
　または隣接した状態でターンエンドボタンを押す、
　またはバトル後にターンエンドとなり、相手のターンとなる。
